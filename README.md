# 📝 NotePad (Qt C++)

A lightweight and user-friendly text editor with offline next word predictions built using the Qt framework.

## 🛠️ Built With

- Qt 6 (C++)
- Qt Widgets

## 📂 Project Structure

```
main.cpp
mainwindow.cpp
mainwindow.h
mainwindow.ui
NotePad.pro
```

## 🚀 How to Run

### Using Qt Creator

1. Open Qt Creator
2. Click **Open Project**
3. Select `NotePad.pro`
4. Build and Run

### Using Terminal (MinGW example)

```
qmake
mingw32-make
./NotePad.exe
```
